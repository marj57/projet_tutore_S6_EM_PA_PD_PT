# Natalie's Cakes

Projet tutoré réalisé dans le cadre de la licence Professionnelle Développeur Web et Mobile pour le Commerce Electronique.

## Objectifs  

Le but de ce projet est de développer une application Web qui permettra de gérer de manière
simple des recettes : édition, ajout et suppression. 

## Contraintes

Développement de l'application avec JavaScript, React ainsi que l'utilisation de Firebase pour le stokage et lecture des données.

## Installation 

Lancer la commande suivante à la racine du dossier 
```
$ npm install
```
Une fois l'installation finie, lancer la commande suivante à la racine du dossier
```
$ npm start
```

## Lancement

L'application devrait s'ouvrir automatiquement sur le navigateur.
